# 📝 To-Do List

## 🎯 Motivação

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de manipulação do DOM e persistência de dados no navegador. A proposta é criar uma aplicação funcional e intuitiva de lista de tarefas (to-do list), com interação via arrastar e soltar (drag and drop).

## 🚀 Funcionalidades

- ✅ Adição e remoção de tarefas
- 🔄 Movimentação de tarefas entre colunas por **drag and drop**
- 💾 Persistência das tarefas via `localStorage`

## 🧰 Tecnologias Utilizadas

- HTML5
- CSS3 (organizado em `/src/styles`)
- JavaScript puro (Vanilla JS – organizado em `/src/js`)
- Imagens e ícones (armazenados em `/src/img`)
